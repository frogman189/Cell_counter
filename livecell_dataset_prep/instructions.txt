Instructions for Preparing the LIVECell Dataset

1. Download the dataset
    Go to https://www.kaggle.com/datasets/markunys/livecell-dataset/data and download the following four files:
        1. LIVECell_dataset_2021.zip
        2. livecell_annotations_train.json.zip
        3. livecell_annotations_val.json.zip
        4. livecell_annotations_test.json.zip

2. Move the files
    Place all four files into your designated working directory.

3. Update script paths
    Open the script `extract_data.py` and edit the dataset file paths inside the `main` function to match the directory where you placed the files.

4. Run the script
    Execute the script `extract_data.py`.
    This will create a new directory containing:
        - All extracted images
        - A CSV file with image information
